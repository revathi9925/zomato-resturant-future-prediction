# 🍽️ Zomato Restaurant Analytics & Future Popularity Prediction

An interactive Streamlit dashboard that analyzes restaurant data and predicts future popularity using data analytics, visualization, and machine learning.

## 🚀 Features

- 📊 **Restaurant performance analytics** (ratings, votes, cost analysis)
- 🔮 **Future popularity prediction** (ML-based trend forecasting)
- 🤖 **ML-based segmentation** (K-Means clustering for restaurant grouping)
- 🏆 **Top performers leaderboard** (identify best restaurants)
- 🎯 **Smart restaurant recommendation system** (personalized suggestions)

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Streamlit** | Interactive web dashboard |
| **Pandas, NumPy** | Data processing & analysis |
| **Plotly** | Interactive visualizations |
| **Scikit-learn** | Machine learning (K-Means, prediction models) |

## 📊 Dataset

- Synthetic dataset generated using NumPy
- Includes: Ratings, Votes, Cost for multiple cuisine types
- Online ordering availability indicator
- 100+ restaurant records for analysis

## 🎯 Use Cases

- Analyze restaurant performance metrics
- Identify high-growth potential restaurants
- Understand market trends and customer preferences
- Demonstrate end-to-end data analytics + ML pipeline
