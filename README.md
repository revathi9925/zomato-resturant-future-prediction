# zomato-resturant-future-prediction
# 🍽️ Zomato Restaurant Analytics & Future Popularity Prediction

An interactive Streamlit dashboard that analyzes restaurant data and predicts future popularity using data analytics, visualization, and machine learning.

## 🚀 Features

- 📊 Restaurant performance analytics (ratings, votes, cost)
- 🔮 Future popularity prediction
- 🤖 ML-based segmentation (K-Means clustering)
- 🏆 Top performers leaderboard
- 🎯 Smart restaurant recommendation system

## 🛠️ Tech Stack

- **Streamlit** - Interactive dashboard
- **Pandas, NumPy** - Data processing
- **Plotly** - Visualizations
- **Scikit-learn** - Machine learning

## 📊 Dataset

- Synthetic dataset generated using NumPy
- Includes: Ratings, Votes, Cost for two cuisine types
- Online ordering availability

## 🎯 Use Cases

- Analyze restaurant performance
- Identify high-growth restaurants
- Understand market trends
- Demonstrate ML + dashboard skills

## 🚀 How to Run

```bash
# Clone the repository
git clone https://github.com/revathi9925/zomato-resturant-future-prediction
cd zomato-restaurant-prediction

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
